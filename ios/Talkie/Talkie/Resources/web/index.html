Politique de confidentialité

Dernière mise à jour : Mars 2026

Résumé

Talkie respecte votre vie privée. Toutes vos données restent sur votre appareil. Aucune donnée personnelle n'est collectée ni transmise à nos serveurs.

Données stockées localement

Les informations suivantes sont stockées uniquement sur votre iPhone :

  • Votre nom (si renseigné)
  • L'historique des conversations (les 20 derniers échanges)
  • Les informations retenues par la mémoire IA
  • Le prompt système personnalisé
  • Les échantillons vocaux (si fournis)

ElevenLabs (optionnel)

Si vous activez la voix ElevenLabs, votre clé API est stockée de manière sécurisée dans le Keychain iOS. Lorsque vous utilisez cette fonctionnalité, le texte de vos réponses est envoyé aux serveurs d'ElevenLabs pour la synthèse vocale. Aucune donnée audio ou textuelle n'est conservée par Talkie. Votre consentement est demandé avant la première activation.

Apple Intelligence

Les suggestions IA sont générées localement via Apple Intelligence sur votre appareil. Aucune donnée de conversation n'est envoyée à des serveurs tiers pour cette fonctionnalité.

Reconnaissance vocale

La reconnaissance vocale utilise l'API Web Speech de votre navigateur intégré. Le traitement peut impliquer les serveurs d'Apple conformément à leur politique de confidentialité.

Suppression des données

Vous pouvez supprimer toutes vos données à tout moment dans Réglages → Données personnelles → Supprimer toutes mes données.

Support

Comment utiliser Talkie

Talkie est une application de communication assistée conçue pour les personnes vivant avec la SLA (maladie de Charcot), les personnes muettes ou ayant des difficultés à parler.

Fonctionnement

  • Écoute : Appuyez sur la barre d'écoute pour activer le micro. Talkie transcrit ce que dit votre interlocuteur.
  • Suggestions : Si Apple Intelligence est activé, 3 réponses sont proposées automatiquement.
  • Parler : Appuyez sur une suggestion pour la prononcer à voix haute.
  • Saisie libre : Utilisez le bouton « Écrire » pour taper manuellement votre message.

Suggestions IA

Les suggestions sont générées par Apple Intelligence directement sur votre appareil. Assurez-vous qu'Apple Intelligence est activé dans Réglages iOS → Apple Intelligence et Siri.

Personnaliser la voix

Par défaut, Talkie utilise la voix intégrée de votre iPhone. Vous pouvez activer une voix personnalisée via ElevenLabs dans les réglages pour une expérience plus naturelle.

Questions fréquentes

Le micro ne fonctionne pas ?
Vérifiez que Talkie a l'autorisation d'accéder au microphone dans Réglages iOS → Talkie.

Les suggestions n'apparaissent pas ?
Assurez-vous qu'Apple Intelligence est activé dans Réglages iOS → Apple Intelligence et Siri.

Comment supprimer mes données ?
Allez dans Réglages → Données personnelles → Supprimer toutes mes données.
