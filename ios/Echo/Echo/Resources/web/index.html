Bienvenue sur Écho
Une application de communication assistée, conçue pour les personnes vivant avec la SLA.
Comment ça marche
1
L'interlocuteur parle. Appuyez sur la barre du haut pour activer l'écoute.
2
Écho transcrit et propose 3 réponses adaptées.
3
Appuyez sur une réponse pour la prononcer à voix haute.
Personnalisez
Dans les réglages, vous pouvez ajouter votre clé ElevenLabs pour une voix clonée, ou utiliser la voix iOS par défaut. Tout est sauvegardé sur votre appareil.
